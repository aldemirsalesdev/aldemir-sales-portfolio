## Uso de Inteligência Artificial

Durante o desenvolvimento deste projeto, utilizei ferramentas de Inteligência Artificial, como o Google AI Studio, como apoio no processo de construção.

A IA foi utilizada para:

- Auxiliar no desenvolvimento do front-end  
- Apoiar na implementação de partes do back-end  
- Ajudar na estruturação de código e resolução de problemas  
- Acelerar o aprendizado durante o desenvolvimento  

Todo o código gerado foi revisado, adaptado e compreendido por mim, fazendo parte do meu processo de evolução prática como desenvolvedor.
